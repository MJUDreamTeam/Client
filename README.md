## 디렉토리 설명

* navigators 폴더

	Navigation bar와 관련된 컴포넌트, Navigation 컨테이너 들이 들어있다.

* tabs 폴더

	하부에 핵심 기능 5가지를 각각 다른 파일로 나누어 놓았다.

* Components/Activity 폴더

	활동 기능에 들어갈 댓글, 좋아요, 랜덤추천, 팔로우 컴포넌트가 들어있다.
